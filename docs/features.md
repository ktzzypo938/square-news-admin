# Square News Admin — 功能簡介

> 本文件列出目前已實作的所有功能模組，供顧問快速了解系統現況。

---

## 功能地圖

```
Square News Admin
├── 公開區（不需登入）
│   ├── 儀表板 Dashboard
│   ├── 事件列表 / 事件詳情
│   ├── 事件煽動分析
│   └── 文章列表 / 文章詳情
│
└── 後台管理區（需 @square.news Google 帳號登入）
    ├── 媒體來源管理
    ├── 會員管理
    ├── 事件聚合管理
    ├── 煽動指數管理
    ├── 實體別名管理（開發中）
    └── 操作面板
```

---

## 公開區功能

### 儀表板（`/dashboard`）
- 關鍵字快速光譜搜尋
- 偏向指數條（偏藍 / 中立 / 偏綠）視覺化
- 趨勢事件卡片牆
- 各媒體文章數量長條圖（ECharts）
- 媒體來源統計摘要

---

### 事件列表（`/events`）
- 分頁表格（每頁 20 筆）
- 依主題關鍵字搜尋
- 篩選器：狀態（OPEN / CLOSED）、發布狀態
- 操作（登入後可用）：發布 / 下架、標記頭條
- 欄位：主題、文章數、熱度、多元性、最後更新時間

### 事件詳情（`/events/:eventId`）
- 事件元資料（ID、主題、狀態、文章數、來源數）
- 熱度 / 立場多元性指標
- 重點摘要、相關人物
- 連結至煽動分析頁

### 事件煽動分析（`/events/:eventId/incitement`）
- 2D 散佈圖：X 軸為立場極性，Y 軸為煽動分數
- 媒體煽動排行榜
- 平均煽動分數 / 平均立場統計
- 各媒體詳細分析（可展開）、信心分數

---

### 文章列表（`/articles`）
- 分頁表格，可調頁面大小（10 / 20 / 50 / 100）
- 篩選：來源媒體、日期區間
- 語意關鍵字搜尋
- 欄位：標題、媒體、發布日期、狀態

### 文章詳情（`/articles/:id`）
- 完整文章資訊
- 煽動分析（若已分析）：
  - 7 維度煽動評分（A–G）
  - 佐證片段與引用類型
  - 立場分析與信心分數

---

## 後台管理區（需登入）

### 媒體來源管理（`/admin/media-sources`）
| 功能 | 說明 |
|------|------|
| 新增媒體 | 名稱、代碼、URL、類別、預設偏向分數、權重、事實性評級、所有權 |
| 編輯媒體 | 更新所有欄位 |
| 刪除媒體 | 移除媒體來源 |
| 偏向色塊 | 依偏向分數（-1 至 +1）顯示顏色 |

**媒體類別**：TRADITIONAL（傳統）/ ONLINE（網路）/ INDEPENDENT（獨立）/ OTHER
**事實性評級**：HIGH / MIXED / LOW / UNKNOWN

---

### 會員管理（`/admin/members`）
- 統計卡片：總數、活躍、停用、已上線
- 依姓名 / Email 搜尋
- 角色與狀態篩選
- 用戶詳情 Modal：頭像、Email、角色、上線狀態、興趣、性別、出生年、最後登入時間

---

### 事件聚合管理（`/admin/event-aggregation`）
- 搜尋 / 篩選事件
- 加入 / 移除事件的文章
- 重新計算事件統計數據
- 重新產生 AI 分析
- 解散事件（將文章重新聚合）
- 批次重新聚合文章

---

### 煽動指數管理（`/admin/incitement-management`）
- 統計儀表板：
  - 總事件數 / 已分析 / 未分析 / 部分分析
- 依分析狀態篩選
- 主題關鍵字即時搜尋（debounce）
- 批次分析按鈕
- 事件表格含分析狀態標籤

---

### 實體別名管理（`/admin/entity-aliases`）
- 開發中，預計支援 PERSON / ORGANIZATION / LOCATION / EVENT 類別的別名管理

---

### 操作面板（`/admin/operations`）
| 功能 | 說明 |
|------|------|
| 新聞搜尋 | 關鍵字搜尋文章、選取後送 AI 產生事件草稿 |
| 手動建立事件 | 主題、地點標籤、盲點標籤、重點列表、核心摘要、參與者 JSON、時間範圍、狀態 |
| 文章批次重聚合 | 批次觸發文章重新聚合 |

---

## 核心業務概念

| 概念 | 說明 |
|------|------|
| 事件（Event） | 由多篇文章聚合而成的新聞議題 |
| 立場偏向 | -1（偏藍/親中）至 +1（偏綠/親美）連續分數 |
| 煽動指數 | 7 個維度（A–G）評估媒體報導的煽動程度 |
| 光譜搜尋 | 依關鍵字查詢各媒體立場分布 |
| 聚合（Aggregation） | 系統自動將相關文章歸入同一事件 |

---

## 目前限制 / 已知狀況

- 實體別名管理尚未完成
- 無前端自動化測試（無 Vitest / Jest 設定）
- 後台操作 timeout 設定為 300 秒（部分 AI 分析任務耗時較長）
